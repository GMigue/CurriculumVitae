SIMULACIÓN ANIMADA DE UN BANCO EN JAVA

Simulación animada de clientes en un banco

Simulación animada de clientes entrando a un banco, esperando su turno, pasando con un cajero para realizar operaciones y una vez terminadas las operaciones proceden a salir del banco.

Este proyecto se realizo usando JAVA usando Sprite para el proceso de la animación y utilizando programación paralela para el movimiento de cada uno de los clientes. El proyecto en funcionamiento se puede observar en el siguiente video.
