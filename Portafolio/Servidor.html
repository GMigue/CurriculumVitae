SERVIDOR EN LINUX Y MONITOREO

Implementación de un servidor web, FTP, DNS, de correo, SSH y creación de una zona desmilitarizada (DMZ) en Linux. Al igual que su posterior monitoreo para verificar fallas en la seguridad del servidor.

Como proyecto de investigación junior para el II CONGRESO INTERNACIONAL DE CIENCIAS DE LA COMPUTACIÓN y el X CONGRESO NACIONAL DE CIENCIAS DE LA COMPUTACIÓN trabaje en la creación de un servidor con las siguientes características:

Servidor Web
Para el servidor web que maneja paginas http y https se uso apache2 junto con OpenSSL para la creación del sitio https con el fin de crear un par de claves y certificado autofirmados.
Servidor FTP
Se utilizo el programa vsFTPD para el servidor FTP y se configuro para acceder a la carpeta del sitio web al cual por seguridad solo se podia acceder con un usuario y contraseña.
Servidor de correo
Se instalo postfix para el correo SMTP y para el poder visualizar y administrar el correo se utilizo dovecot-imap, squirrelmail y apache2 para crear una aplicación webmail.
Servidor SSH
Para poder acceder remotamente al servidor se creo un servidor SSH utilizando el programa OpenSSH.
Servidor DNS
En el caso del servidor DNS se utilizo Bind9. Se configuro como DNS maestro y se agrego los servicios http, https, ftp, ssh, smtp e imap.
Zona desmilitarizada (DMZ)
Para mejorar la seguridad del servidor y en especial los equipos que se encuentran en la red local se creo una zona desmilitarizada (DMZ). Esta fue configurada a través de iptables y redireccionando las peticiones del exterior a la DMZ.

Una vez teniendo en funcionamiento el servidor se utilizo NMAP para ejecutar un script de vulnerabilidades al servidor, encontrando una vulnerabilidad de fuerza de grupo DH insuficiente en el intercambio de claves Diffie-Hellman en el puerto 25 (smtp).

Teniendo identificada la vulnerabilidad se creo una clave DH de 2048 bits con OpenSSL y se modifico la configuración de postfix para que usara el DH de 2048 y que evitara usar otros cifrados. Con lo anterior se corrigió el problema detectado.
