1BIT (APP MOVIL CON FLUTTER)

Desarrollo de agenda dinámica en android usando el framework Flutter y Firebase

Desarrollo de una aplicación móvil para android usando el framework de Flutter junto con la base de datos de Firebase para el almacenamiento de la información.

El objetivo de la aplicación es el almacenamiento de eventos y tareas pudiendo asignar a estas ultimas un horario de manera automática para poder realizarlas de acuerdo a la prioridad asignada, el solapamiento con otras tareas o eventos y una seria de restricciones con respecto a horarios en los que poder realizarlas.

Flutter, Firebase
