GRÁFICAS CON D3.JS, ECHART.JS, MySQL Y PHP

Creación de gráficas personalizadas a partir de filtros con información de la SEP de estudiantes foráneos de nivel licenciatura

BASE DE DATOS

A partir de información obtenida de la SEP en formato EXCEL de la residencia de estudiantes de nivel licenciatura y de la ubicación de las universidades, su oferta educativa y la cantidad de alumnos en cada carrera se realizo la importación a una base de datos en MySQL

API EN PHP

Teniendo en información en MySQL se creo una API con PHP para poder consultar la información teniendo en cuenta que cada gráfica cuenta con una serie de filtros que se le pueden aplicar

ECHART GRÁFICA DE PIRÁMIDE

Se realizo 4 gráficas de tipo pirámide poblacional con la librería ECHART en donde se representa la cantidad de alumnos por sexo y divididas en:

  • Entidad de las universidades
  • Municipio de las universidades
  • Por universidades
  • Por carreras

Entre los filtros que se pueden aplicar están:

  • Matriculas totales o solo de primer ingreso
  • Carreras
  • Estado
  • Municipio
  • Nivel (licenciatura o carrera técnica)
  • Sostenimiento
  • Instituciones
  • Año de los datos
  • Controles para la animación de la gráfica por años

ECHART GRÁFICA SANKEY

Se realizo 2 gráficas de tipo sankey con la librería ECHART en donde se representa los estados de procedencia de los alumnos y las universidades en donde se encuentra matriculados. Estas dos gráficas se encuentran divididas en:

  • Universidades
  • Carreras ofertadas por las universidades

Entre los filtros que se pueden aplicar están:

  • Carreras
  • Estado de las universidades
  • Municipio de las universidades
  • Nivel (licenciatura o carrera técnica)
  • Sostenimiento
  • Instituciones
  • Año de los datos

D3 GRÁFICA DE PESAS

Se realizo una gráfica de pesas con la librería D3 en donde se representa el crecimiento de las matriculas en los programas de las universidades entre dos años seleccionados

  • Gráfica de pesas

Entre los filtros que se pueden aplicar están:

  • Ordenamiento de los datos
  • Matriculas totales o solo de primer ingreso
  • Selección de los años a comparar
  • Estado de las universidades
  • Municipio de las universidades
  • Nivel (licenciatura o carrera técnica)
  • Sostenimiento
  • Instituciones
  • Carreras
  • Programas

MySQL, PHP, JavaScript, D3.js, Echart.js, CCS, HTML
